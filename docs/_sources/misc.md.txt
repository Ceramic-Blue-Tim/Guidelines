# Miscellaneous

## Password managing

If you feel skeptical about the use usufeulness and convenience of a password manager, then you may document yourself on the matter. Password managers do are something you want to get into. The one I recommend is KeePass.

Information on the software : [here](https://keepass.fr/)

Download page : [here](https://keepass.info/download.html)

Since it is cross-platform and works on a light database file, you could think of a cloud hosted database used to refresh and update all local databases on your device. I recommend [Nextcloud](https://nextcloud.com/) as a self-hosted cloud but other alternatives sure exist.
